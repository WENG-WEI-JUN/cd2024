解釋: 導入庫: 除了你的 PID 控制代碼所需的庫之外，我們還需要導入 matplotlib.pyplot 來繪制圖形。 創建繪圖窗口和子圖: 使用 plt.subplots(2, 1) 創建一個包含兩個子圖的繪圖窗口，分別用於繪制 x 和 y 方向的位置。 設置標題: 使用 ax.set_title() 設置每個子圖的標題。 初始化數據列表: 創建空列表 x_data 和 y_data，用於存儲鋼球的位置數據。 控制循環: 在你的 PID 控制邏輯中，添加以下步驟： 獲取鋼球坐標: 使用 vrep.simxGetObjectPosition() 獲取鋼球在 x 和 y 方向的位置。 添加數據到列表: 將獲取到的 x 和 y 坐標添加到 x_data 和 y_data 列表中。 更新繪圖: 使用 ax.plot() 繪制 x 和 y 方向的位置數據。 調整坐標軸範圍: 使用 ax.relim() 和 ax.autoscale_view() 調整坐標軸範圍，以適應不斷增加的數據。 暫停顯示圖形: 使用 plt.pause(0.01) 暫停程序 0.01 秒，以便顯示更新後的圖形。 關閉連接: 在控制循環結束後，使用 vrep.simxFinish() 關閉與 CoppeliaSim 的連接。 注意: 你需要將 ...
